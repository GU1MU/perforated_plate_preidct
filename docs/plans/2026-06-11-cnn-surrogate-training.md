# CNN Surrogate Training Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add one configurable CNN training workflow that learns the mapping from 24-hole center layouts to \(K^*\) and \(K_{\varepsilon,\max}\) using the ODB extraction table.

**Architecture:** The training script lives under `3_CNN/scripts/`, reads the upstream FEM extraction table from `2_FEM/results/odb_ml_data/odb_ml_summary.csv`, converts each sample's 24 hole centers into a \(1\times80\times40\) binary image, splits data per group using a configurable training instance count, trains a small PyTorch multi-output CNN, writes CNN-owned outputs under `3_CNN/results/` and `3_CNN/figures/`, and optionally saves the model package when `SAVE_MODEL=True`. The implementation keeps all tuning parameters as top-level constants, so the user can edit one script without command-line arguments.

**Tech Stack:** Python 3, pandas, numpy, PyTorch, scikit-learn, matplotlib, pickle from the standard library, unittest.

`TRAIN_TEST_SPLIT` means:

- For each `group_index`, use `TRAIN_TEST_SPLIT` instances for training.
- Split all remaining instances in that group as evenly as possible between validation and test.
- If the remaining count is odd, assign the extra sample to validation.
- If a group has `count <= TRAIN_TEST_SPLIT`, place all available samples from that group into training and leave validation/test empty for that group.
- Never hard-code the total dataset size, the number of groups, or fixed train/validation/test counts.

`SAVE_MODEL` means:

- If `SAVE_MODEL=True`, save `model.pt` and `target_scaler.pkl`.
- If `SAVE_MODEL=False`, do not save `model.pt` or `target_scaler.pkl`.
- Other diagnostic outputs are still written because they document the run and do not imply model reuse.

## 5. Image Encoding

Convert each sample to a single-channel binary image:

$$
X\in\{0,1\}^{1\times80\times40}
$$

Use 2 mm per pixel:

$$
r=\left\lfloor\frac{y}{2}\right\rfloor,\qquad c=\left\lfloor\frac{x}{2}\right\rfloor
$$

Clamp indices into valid image bounds:

$$
0\le r<80,\qquad 0\le c<40
$$

Encoding convention:

- Hole-center pixel: `1`
- Background: `0`
- Do not draw filled circular holes in this first version.

## 7. Tuning Method And Paths

Use `3_CNN/scripts/train_cnn_surrogate.py` as the single tuning surface.

Tune parameters in this order:

1. `TRAIN_TEST_SPLIT`: increase as more extracted ODB rows become available. Keep validation and test non-empty once the dataset is large enough.
2. `LEARNING_RATE`: start at \(10^{-3}\). If validation loss oscillates, try \(3\times10^{-4}\). If loss decreases too slowly, try \(2\times10^{-3}\).
3. `DROPOUT`: start at 0.2. If train loss improves but validation loss worsens, try 0.3 or 0.4.
4. `WEIGHT_DECAY`: start at \(10^{-4}\). If overfitting persists, try \(10^{-3}\).
5. `BATCH_SIZE`: start at 32. If the dataset is small, try 16. If the dataset grows and GPU/CPU memory permits, try 64.
6. `EPOCHS`: start at 300. Use `train_history.csv` and `loss_curve.png` to decide whether to shorten or extend.

Use these paths for comparison between tuning runs:

- `3_CNN/results/cnn_surrogate/metrics.json`
- `3_CNN/results/cnn_surrogate/predictions.csv`
- `3_CNN/figures/cnn_surrogate/loss_curve.png`
- `3_CNN/figures/cnn_surrogate/pred_vs_true.png`

If preserving multiple tuning runs is needed later, add a `RUN_ID` parameter in a separate change. Do not add it in the first implementation.

```python
    columns = ["odb_name", "status", "group_index", "instance_index"]

def validate_columns(frame):
    missing = [column for column in required_columns() if column not in frame.columns]
    if missing:
        raise ValueError("missing required columns: %s" % ", ".join(missing))
    return None


def load_dataset_table(path=DATA_CSV):
    frame = pd.read_csv(path)
    validate_columns(frame)
    frame = filter_valid_rows(frame)
    frame = frame.dropna(subset=required_columns())
    frame["group_index"] = frame["group_index"].astype(int)
    frame["instance_index"] = frame["instance_index"].astype(int)
```

```python
def assign_splits(frame, train_count=TRAIN_TEST_SPLIT, shuffle=SPLIT_SHUFFLE, random_seed=RANDOM_SEED):
    pieces = []
    rng = np.random.RandomState(random_seed)
    for _, group in frame.groupby("group_index"):
        group = group.sort_values("instance_index").copy()
        indices = list(group.index)
        if shuffle:
            rng.shuffle(indices)
        split_by_index = {}
        train_indices = indices[:train_count]
        remaining = indices[train_count:]
        val_count = int(math.ceil(len(remaining) / 2.0))
        val_indices = remaining[:val_count]
        test_indices = remaining[val_count:]
        for index in train_indices:
            split_by_index[index] = "train"
        for index in val_indices:
            split_by_index[index] = "val"
        for index in test_indices:
            split_by_index[index] = "test"
        group["split"] = [split_by_index[index] for index in group.index]
        pieces.append(group)
    return pd.concat(pieces, axis=0).sort_values(["group_index", "instance_index"]).reset_index(drop=True)
```

```python
def _clamp(value, low, high):
    return max(low, min(high, value))


def encode_hole_image(row, pixel_size=PIXEL_SIZE, height=IMAGE_HEIGHT, width=IMAGE_WIDTH):
    image = np.zeros((1, height, width), dtype=np.float32)
    for index in range(1, 25):
        x = float(row["hole_%02d_x" % index])
        y = float(row["hole_%02d_y" % index])
        r = _clamp(int(math.floor(y / pixel_size)), 0, height - 1)
        c = _clamp(int(math.floor(x / pixel_size)), 0, width - 1)
        image[0, r, c] = 1.0
    return image
```

```python
class HoleLayoutDataset(Dataset):
    def __init__(self, frame, target_scaler=None, fit_scaler=False):
        self.frame = frame.reset_index(drop=True)
        self.images = np.stack([encode_hole_image(row) for _, row in self.frame.iterrows()])
        targets = self.frame[TARGET_COLUMNS].values.astype(np.float32)
        if target_scaler is None:
            target_scaler = StandardScaler()
        if fit_scaler:
            targets = target_scaler.fit_transform(targets)
        else:
            targets = target_scaler.transform(targets)
        self.targets = targets.astype(np.float32)
        self.target_scaler = target_scaler

    def __len__(self):
        return len(self.frame)

    def __getitem__(self, index):
        return torch.from_numpy(self.images[index]), torch.from_numpy(self.targets[index])


class CnnSurrogate(nn.Module):
    def __init__(self, dropout=DROPOUT):
        super(CnnSurrogate, self).__init__()
        self.features = nn.Sequential(
            nn.Conv2d(1, 16, kernel_size=3, padding=1),
            nn.BatchNorm2d(16),
            nn.ReLU(),
            nn.Conv2d(16, 32, kernel_size=3, padding=1),
            nn.BatchNorm2d(32),
            nn.ReLU(),
            nn.MaxPool2d(2),
            nn.Conv2d(32, 64, kernel_size=3, padding=1),
            nn.BatchNorm2d(64),
            nn.ReLU(),
            nn.MaxPool2d(2),
            nn.AdaptiveAvgPool2d((1, 1)),
        )
        self.regressor = nn.Sequential(
            nn.Flatten(),
            nn.Linear(64, 64),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(64, 2),
        )

    def forward(self, inputs):
        return self.regressor(self.features(inputs))
```
